説明

1.1 ハッシュテーブルとは

ハッシュテーブルは配列（スロット）とハッシュ関数の組み合わせです。

概念
キー "apple"  → ハッシュ関数 → インデックス 0 → テーブル[0] に格納
キー "banana" → ハッシュ関数 → インデックス 9 → テーブル[9] に格納
  • キーからハッシュ関数でインデックスを計算し、そのインデックスにデータを格納する
  • 検索・格納が平均 O(1) で行える高速なデータ構造

1.2 衝突（Collision）とは

異なるキーが同じインデックスに格納されようとすること。

  • テーブルサイズは有限（例: 10 スロット）
  • 格納するキーは無限に存在しうる
  • 鳩の巣原理: 11個のデータを10個のスロットに入れたら、必ずどこかが重複する

1.3 線形探索法（Linear Probing）

駐車場を想像してください。目的の番号の駐車場が埋まっていたら、隣の空いている場所に停めます。探すときも同じルートで探します。

衝突が起きたら次のスロットを試す。それも埋まっていたらさらに次へ。

1.4 チェイン法（Chaining）

各スロットにリストを持たせ、衝突したデータをリストに追加する方法。

概念
テーブル[3] → [("apple", 100), ("pleap", 999)]
テーブル[9] → [("banana", 80)]
比較線形探索法チェイン法
衝突時空きスロットを探すリストに追加
テーブル満杯時性能低下格納可能
メモリ配列のみリストの追加メモリ

1.5 Python の dict

Python の dict は内部的にハッシュテーブルを使っています。hash() 組み込み関数でハッシュ値を確認できます。

Python
print(hash("apple"))   # Pythonが計算するハッシュ値
print(hash("banana"))  # 毎回同じ値（決定性）

例題 1

配列をテーブルとして使う

サイズ10の配列を作成し、ハッシュ関数でインデックスを計算してデータを格納します。

Python
table_size = 10
table = [None] * table_size  # 全スロットをNoneで初期化

def hash_function(key, size):
    total = 0
    for char in key:
        total = total + ord(char)
    return total % size

key = "apple"
value = 100
index = hash_function(key, table_size)
print(f"'{key}' のハッシュ値: {index}")

table[index] = (key, value)
print(f"テーブル[{index}] に ('{key}', {value}) を格納しました")
例題 2

ハッシュテーブルへのデータ挿入

複数のデータをハッシュテーブルに挿入し、テーブルの状態を確認します。

Python
def insert_item(table, key, value):
    size = len(table)
    index = hash_function(key, size)
    print(f"  '{key}' → インデックス {index}", end="")
    if table[index] is None:
        table[index] = (key, value)
        print(f" → 格納完了")
    else:
        print(f" → 既にデータあり: {table[index]}")

table = [None] * 10
insert_item(table, "apple", 100)
insert_item(table, "banana", 80)
insert_item(table, "cherry", 150)
insert_item(table, "date", 200)

keys = ["apple", "pleap", "banana", "ananab", "cherry"]

例題 4

線形探索法で格納と検索

Python
def insert_with_probing(table, key, value):
    size = len(table)
    index = hash_function(key, size)
    original_index = index
    probe_count = 0
    while table[index] is not None:
        if table[index][0] == key:
            break
        probe_count += 1
        index = (index + 1) % size
        if index == original_index:
            print("テーブルが満杯です")
            return -1
    table[index] = (key, value)
    print(f"  insert '{key}': 初期={original_index}, 探索{probe_count}回 → [{index}]")

def search_with_probing(table, key):
    size = len(table)
    index = hash_function(key, size)
    start_index = index
    probe_count = 0
    while table[index] is not None:
        probe_count += 1
        if table[index][0] == key:
            print(f"  search '{key}': [{index}]で発見, 探索{probe_count}回")
            return table[index][1]
        index = (index + 1) % size
        if index == start_index:
            break
    print(f"  search '{key}': 見つかりません")
    return None

table = [None] * 10
insert_with_probing(table, "apple", 100)
insert_with_probing(table, "pleap", 999)  # appleと衝突
search_with_probing(table, "apple")
search_with_probing(table, "pleap")
例題 5

HashTable クラスの作成

insert / search / delete / display メソッドを持つクラスを作成します。

Python
class HashTable:
    def __init__(self, size=10):
        self.size = size
        self.table = [None] * size
        self.count = 0

    def _hash(self, key):
        total = 0
        for char in key:
            total += ord(char)
        return total % self.size

    def insert(self, key, value):
        index = self._hash(key)
        original = index
        while self.table[index] is not None:
            if self.table[index][0] == key:
                break
            index = (index + 1) % self.size
            if index == original:
                return -1
        self.table[index] = (key, value)
        self.count += 1

    def search(self, key):
        index = self._hash(key)
        start = index
        while self.table[index] is not None:
            if self.table[index][0] == key:
                return self.table[index][1]
            index = (index + 1) % self.size
            if index == start: break
        return None

    def delete(self, key):
        index = self._hash(key)
        start = index
        while self.table[index] is not None:
            if self.table[index][0] == key:
                self.table[index] = None
                self.count -= 1
                return True
            index = (index + 1) % self.size
            if index == start: break
        return False

# --- テスト ---
ht = HashTable(10)
ht.insert("apple", 100)
ht.insert("banana", 80)
print(ht.search("apple"))   # → 100
ht.delete("apple")
print(ht.search("apple"))   # → None
