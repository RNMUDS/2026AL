計算量を求める

コードのループ構造から計算量をO記法で求めます。

計算量のパターン

パターン計算量コード例
ループなしO(1)value = data[0]
1重ループO(n)for i in range(n): ...
2重ループO(n^2)for i: for j: ...
半分にするO(log n)while n > 1: n //= 2
Python
# --- 実測による確認 ---
sizes = [1000, 2000, 4000, 8000]

for n in sizes:
    data = list(range(n))

    # O(n) の処理
    start_time = time.perf_counter()
    total = 0
    for val in data:
        total += val
    time_n = time.perf_counter() - start_time

    # O(n^2) の処理
    start_time = time.perf_counter()
    count = 0
    for i in range(n):
        for j in range(n):
            count += 1
    time_n2 = time.perf_counter() - start_time

# nが2倍 → O(n)は約2倍、O(n^2)は約4倍になる

例題 4

同じ問題を2つのアルゴリズムで解く

「リスト内の重複要素を見つける」問題を2通りで解き、性能を比較します。

Python
def find_duplicates_bruteforce(data):
    """総当たり版 O(n^2)"""
    duplicates = []
    for i in range(len(data)):
        for j in range(i + 1, len(data)):
            if data[i] == data[j] and data[i] not in duplicates:
                duplicates.append(data[i])
    return duplicates

def find_duplicates_hashset(data):
    """ハッシュセット版 O(n)"""
    seen = set()
    duplicates = set()
    for item in data:
        if item in seen:
            duplicates.add(item)
        else:
            seen.add(item)
    return list(duplicates)

実世界の問題にアルゴリズムを適用する

「図書館の蔵書検索システム」を設計・実装する例です。

機能使うアルゴリズム計算量理由
ID検索ハッシュ探索（辞書）O(1)キーから値を即座に取得
著者検索ハッシュ探索（辞書）O(1)著者名→書籍リストの対応

Python
books = [
    {"id": 1, "title": "Python入門", "author": "田中太郎", "year": 2024},
    {"id": 2, "title": "アルゴリズム基礎", "author": "鈴木花子", "year": 2023},
    # ...
]

# ハッシュテーブルでID検索用インデックスを作る
id_index = {}
for book in books:
    id_index[book["id"]] = book  # O(1) で検索可能に

# 著者名で検索用インデックスを作る
author_index = {}
for book in books:
    author = book["author"]
    if author not in author_index:
        author_index[author] = []
    author_index[author].append(book)

標準課題 2

総合トレース：探索＋ソート＋辞書の組み合わせ

以下のコードを実行せずに、各ステップの変数変化と最終出力を予測しなさい。
予測を書いた後に、AL15-3.py として保存・実行して確かめなさい。

Python ── AL15-3.py
data = [4, 2, 7, 1, 5, 3, 6]

# Step 1: 挿入ソート
for i in range(1, len(data)):
    key = data[i]
    j = i - 1
    while j >= 0 and data[j] > key:
        data[j + 1] = data[j]
        j -= 1
    data[j + 1] = key
print(f"ソート後: {data}")

# Step 2: 二分探索
target = 5
left, right = 0, len(data) - 1
count = 0
while left <= right:
    mid = (left + right) // 2
    count += 1
    if data[mid] == target:
        break
    elif data[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
print(f"target={target}: index={mid}, 比較{count}回")

# Step 3: 辞書で集計
groups = {}
for d in data:
    key = "偶数" if d % 2 == 0 else "奇数"
    if key not in groups:
        groups[key] = []
    groups[key].append(d)
print(f"分類: {groups}")

ヒント: 3つのステップを順番に追います。Step 1 のソート結果が Step 2 の入力になります。Step 3 は独立した処理です。
