説明

1.1 これまでのソートとの違い

これまで学んだソートアルゴリズム（挿入ソート、バブルソート）は要素同士を比較して順序を決めていました。比較ベースのソートの下限は O(n log n) です。

分布数えソート（Counting Sort）は比較を一切使いません。代わりに、各値が何回出現するかを数えることでソートします。

方式アプローチ計算量
比較ベース「AとBはどちらが大きい？」を繰り返す最速でも O(n log n)
分布数えソート「値0は何個？値1は何個？...」と数えて並べ直すO(n + k)

1.2 分布数えソートの3ステップ

テストの答案用紙を点数別に仕分けする作業を想像してください。 まず点数ごとの山を作り（度数を数える）、次に「60点以下は何人？」と累計を出し（累積度数）、最後に答案を正しい順番に並べる（配置）。

1.3 なぜ末尾から処理するのか？

末尾から処理すると、同じ値の要素の元の順序が保たれます（安定ソート）。これは次回（第11回）で詳しく学びます。

1.4 計算量

操作計算量
度数を数えるO(n)
累積度数を計算O(k)
要素を配置O(n)
合計O(n + k)
  • n: 要素数
  • k: 値の範囲（最大値 - 最小値 + 1）
  • k が小さければ（例: テストの点数 0~100）、非常に高速

キーワード解説

用語説明
度数分布
(Counting Sort)		各値の出現回数を数えてソートする方法。要素同士を比較せず、カウントだけで順序を決定する。
累積度数度数を前から足し合わせた配列。「値x以下の要素が何個あるか」を表し、出力位置の決定に使う。
非比較ソート要素同士の大小比較を使わないソート。比較ベースの下限 O(n log n) を突破し、O(n+k) を実現できる。
例題 1

リスト内の各値の出現回数を数える

Python
def count_occurrences(data):
    max_val = max(data)
    count = [0] * (max_val + 1)  # 度数配列を0で初期化

    for value in data:
        count[value] += 1  # 該当する値のカウントを増やす

    return count

data = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3]
frequency = count_occurrences(data)

print(f"入力データ: {data}")
print(f"度数配列:   {frequency}")
for i, cnt in enumerate(frequency):
    if cnt > 0:
        print(f"  値 {i}: {cnt} 回")

例題 2

度数配列から累積度数配列を作る

Python
def build_cumulative(count):
    cumulative = count[:]  # 度数配列をコピー
    for i in range(1, len(cumulative)):
        cumulative[i] = cumulative[i] + cumulative[i - 1]
    return cumulative

data = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3]
max_val = max(data)
count = [0] * (max_val + 1)
for value in data:
    count[value] += 1

cumulative = build_cumulative(count)

print(f"度数配列: {count}")
print(f"累積度数: {cumulative}")
for i in range(len(cumulative)):
    if count[i] > 0:
        print(f"  値 {i} 以下の要素数: {cumulative[i]}")

例題 3

基本的な分布数えソート

Python
def counting_sort_basic(data):
    if len(data) <= 1:
        return data[:]

    max_val = max(data)

    # Step 1: 度数を数える
    count = [0] * (max_val + 1)
    for value in data:
        count[value] += 1

    # Step 2: 累積度数を計算する
    for i in range(1, len(count)):
        count[i] += count[i - 1]

    # Step 3: 出力配列に配置する（末尾から = 安定ソート）
    output = [0] * len(data)
    for i in range(len(data) - 1, -1, -1):
        value = data[i]
        position = count[value] - 1  # 配置位置（0始まり）
        output[position] = value
        count[value] -= 1

    return output

data = [4, 2, 2, 8, 3, 3, 1]
result = counting_sort_basic(data)
print(f"入力: {data}")
print(f"出力: {result}")  # → [1, 2, 2, 3, 3, 4, 8]

assert result == sorted(data)
print("検証OK: sorted() と同じ結果")

例題 5

分布数えソートと挿入ソートの速度比較

値の範囲が 0~100 のとき、分布数えソートは挿入ソートに比べて圧倒的に高速です。

Python
import time, random

def counting_sort(data, max_val):
    count = [0] * (max_val + 1)
    for v in data: count[v] += 1
    for i in range(1, len(count)): count[i] += count[i-1]
    output = [0] * len(data)
    for i in range(len(data)-1, -1, -1):
        output[count[data[i]]-1] = data[i]
        count[data[i]] -= 1
    return output

def insertion_sort(data):
    arr = data[:]
    for i in range(1, len(arr)):
        key = arr[i]; j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j+1] = arr[j]; j -= 1
        arr[j+1] = key
    return arr

random.seed(42)
sizes = [1000, 5000, 10000]
max_val = 100

print(f"{'サイズ':>8} {'分布数え':>14} {'挿入ソート':>12} {'速度比':>8}")
for size in sizes:
    data = [random.randint(0, max_val) for _ in range(size)]

    start = time.time()
    counting_sort(data, max_val)
    ct = time.time() - start

    start = time.time()
    insertion_sort(data)
    it = time.time() - start

    ratio = it / ct if ct > 0 else 0
    print(f"{size:>8} {ct:>13.4f}s {it:>11.4f}s {ratio:>7.1f}x")

標準課題 2

分布数えソートコードのトレース

以下のコードを実行せずに、各ステップで freq と output がどう変化するかを追い、最終出力を予測しなさい。
予測を書いた後に、AL10-3.py として保存・実行して確かめなさい。

Python ── AL10-3.py
data = [2, 5, 3, 0, 2, 3, 0, 3]
MAX_VAL = 6

# Step 1: 度数配列
freq = [0] * MAX_VAL
for d in data:
    freq[d] += 1
print(f"度数: {freq}")

# Step 2: 累積度数
for i in range(1, MAX_VAL):
    freq[i] += freq[i - 1]
print(f"累積: {freq}")

# Step 3: 配置（末尾から）
output = [0] * len(data)
for d in reversed(data):
    freq[d] -= 1
    output[freq[d]] = d
print(f"結果: {output}")

ヒント: Step 1 で各値の出現回数を数え、Step 2 で累積し、Step 3 では末尾から配置します。freq の変化を1つずつ追いましょう。
