説明

1.1 CSVファイルとは

CSV（Comma-Separated Values）は、データをカンマで区切ったテキストファイルです。Excel やデータベースのデータをやり取りするのによく使われます。

CSV
名前,数学,英語,国語
田中太郎,85,72,90
佐藤花子,92,88,76
鈴木一郎,78,95,83

1.2 Python での CSV 読み書き

Python には csv モジュールが標準で用意されています。

Python
import csv

# --- 書き込み ---
with open("data.csv", "w", newline="", encoding="utf-8") as f:
    writer = csv.writer(f)
    writer.writerow(["名前", "点数"])    # ヘッダー行
    writer.writerow(["田中", 85])        # データ行
    writer.writerow(["佐藤", 92])

# --- 読み込み ---
with open("data.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    header = next(reader)       # 1行目（ヘッダー）を読み飛ばす
    for row in reader:
        name = row[0]           # 文字列
        score = int(row[1])     # 数値に変換
        print(f"{name}: {score}点")
CSVはExcelの表を「テキストファイル」にしたもの。メモ帳でも開けるシンプルなデータ形式です。プログラム間でデータをやり取りするとき、一番よく使われる形式の1つです。

1.3 安定ソート（Stable Sort）とは

安定ソートとは、同じ値の要素が元の配列での順序を保つソートのことです。

1.4 安定ソートが重要な場面

複数のキーでソートしたい場合に安定ソートが必要です。

例: 成績データを「数学の点数 → 名前の五十音順」で並べたい場合

  1. まず名前の五十音順でソート
  2. 次に数学の点数で安定ソート

→ 同じ点数の人は五十音順が保たれる！

1.5 各ソートアルゴリズムの安定性

アルゴリズム安定？理由
挿入ソート安定「より大きい」場合のみ交換
バブルソート安定「より大きい」場合のみ交換
分布数えソート安定末尾から配置すれば安定
選択ソート不安定離れた要素を交換する
Python sorted()安定Timsort は安定ソート

1.6 基数ソート（Radix Sort）入門

基数ソートは、数値を桁ごとに分布数えソートするアルゴリズムです。

  • 各桁のソートには安定ソートを使う必要がある（だから分布数えソートを使う）
  • 計算量: O(d * (n + k))（d: 最大桁数、n: 要素数、k: 基数（通常10））

import csv  # CSVモジュールを読み込む

def write_scores_csv(filename, data):
    """成績データをCSVファイルに書き出す"""
    with open(filename, "w", newline="", encoding="utf-8") as f:
        writer = csv.writer(f)
        writer.writerow(["name", "math", "english"])  # ヘッダー
        for row in data:
            writer.writerow(row)
    print(f"CSVファイルを保存しました: {filename}")

def read_scores_csv(filename):
    """CSVファイルから成績データを読み込む"""
    records = []
    with open(filename, "r", encoding="utf-8") as f:
        reader = csv.reader(f)
        header = next(reader)  # ヘッダーを読み飛ばす
        for row in reader:
            name = row[0]
            math_score = int(row[1])
            english_score = int(row[2])
            records.append([name, math_score, english_score])
    print(f"読み込み完了: {len(records)} 件")
    return records

# --- 実行 ---
sample_data = [
    ["Tanaka", 85, 72],
    ["Sato", 92, 88],
    ["Suzuki", 78, 95],
    ["Yamada", 65, 80],
]

write_scores_csv("scores.csv", sample_data)
loaded = read_scores_csv("scores.csv")
for record in loaded:
    print(f"  {record[0]}: 数学={record[1]}, 英語={record[2]}")

import csv
import random

def counting_sort_records(records, key_index, max_val):
    """レコードのリストを指定キーで分布数えソートする"""
    count = [0] * (max_val + 1)  # 度数配列を初期化
    for record in records:
        count[record[key_index]] += 1

    for i in range(1, len(count)):  # 累積度数を計算
        count[i] += count[i - 1]

    output = [None] * len(records)  # 出力配列
    for i in range(len(records) - 1, -1, -1):  # 末尾から配置
        key = records[i][key_index]
        position = count[key] - 1
        output[position] = records[i]
        count[key] -= 1

    return output

安定ソートの動作を確認する

Python
def counting_sort_stable(data, key_func, max_val):
    """安定な分布数えソート（キー関数版）"""
    count = [0] * (max_val + 1)
    for item in data:
        count[key_func(item)] += 1

    for i in range(1, len(count)):
        count[i] += count[i - 1]

    output = [None] * len(data)
    for i in range(len(data) - 1, -1, -1):  # 末尾から配置（安定のポイント）
        key = key_func(data[i])
        position = count[key] - 1
        output[position] = data[i]
        count[key] -= 1

    return output

# --- テストデータ ---
students = [
    ("Sato", 80), ("Tanaka", 90),
    ("Suzuki", 80), ("Yamada", 90),
    ("Ito", 80), ("Watanabe", 90),
]

result = counting_sort_stable(students, key_func=lambda s: s[1], max_val=100)

score_80 = [name for name, score in result if score == 80]
print(f"80点の順序: {score_80}")
print(f"元の順序保持: {score_80 == ['Sato', 'Suzuki', 'Ito']}")

import random
import time

def counting_sort_large(data, max_val):
    """大規模データ用の分布数えソート"""
    count = [0] * (max_val + 1)
    for value in data:
        count[value] += 1
    for i in range(1, len(count)):
        count[i] += count[i - 1]
    output = [0] * len(data)
    for i in range(len(data) - 1, -1, -1):
        output[count[data[i]] - 1] = data[i]
        count[data[i]] -= 1
    return output

# --- 大規模データで速度計測 ---

基数ソートの基本実装

Python
def counting_sort_by_digit(data, digit_pos):
    """指定した桁で分布数えソートする"""
    base = 10
    count = [0] * base
    for value in data:
        digit = (value // digit_pos) % base  # 指定桁の数字を取り出す
        count[digit] += 1
    for i in range(1, base):
        count[i] += count[i - 1]
    output = [0] * len(data)
    for i in range(len(data) - 1, -1, -1):
        digit = (data[i] // digit_pos) % base
        position = count[digit] - 1
        output[position] = data[i]
        count[digit] -= 1
    return output

def radix_sort(data):
    """基数ソート: 各桁で分布数えソートを繰り返す"""
    if len(data) <= 1:
        return data[:]
    max_val = max(data)
    result = data[:]
    digit_pos = 1  # 1の位からスタート
    while digit_pos <= max_val:
        print(f"  {digit_pos}の位でソート: {result[:10]}...")
        result = counting_sort_by_digit(result, digit_pos)
        digit_pos *= 10  # 次の桁に進む
    return result

# --- 実行 ---
test_data = [170, 45, 75, 90, 802, 24, 2, 66]
sorted_result = radix_sort(test_data)
print(f"結果: {sorted_result}")

# ===== 発展課題: 基数ソートの仕組みを理解する =====

def counting_sort_by_digit(data, exp):
    """指定した桁（exp）で分布数えソートを行う（安定ソート）"""
    n = len(data)
    output = [0] * n
    count = [0] * 10  # 0〜9の10種類

    # 各要素の該当桁の値をカウント
    for val in data:
        digit = (val // exp) % 10
        count[digit] += 1

    # 累積和（安定性を保つため）
    for i in range(1, 10):
        count[i] += count[i - 1]

    # 後ろから走査して出力配列に配置（安定性を保つ）
    for i in range(n - 1, -1, -1):
        digit = (data[i] // exp) % 10
        count[digit] -= 1
        output[count[digit]] = data[i]

    return output

def radix_sort(data):
    """基数ソート: 各桁ごとに分布数えソートを適用"""
    if not data:
        return data

    max_val = max(data)
    result = data[:]

    exp = 1  # 1の位から開始
    step = 1

    while max_val // exp > 0:
        print(f"--- 第{step}ステップ: {exp}の位でソート ---")
        result = counting_sort_by_digit(result, exp)
        print(f"  結果: {result}")
        exp *= 10
        step += 1

    return result

# --- テスト ---
data = [170, 45, 75, 90, 802, 24, 2, 66]

print("=== 基数ソート(Radix Sort) ===")
print(f"ソート前: {data}")
print()

sorted_data = radix_sort(data)

print()
print(f"ソート後: {sorted_data}")

評価ポイント（これが説明できていればOK）:

  • 基数ソートは「1の位 → 10の位 → 100の位」と下位桁から順にソートする方法
  • 各桁のソートに分布数えソート（安定ソート）を使用し、前の桁のソート結果を壊さない
  • 安定ソートでないと前の桁の順序が崩れるため、基数ソートは「安定ソートの組み合わせ」で成り立っている
  • 計算量は O(d * (n + k)) で、d は最大桁数、k は基数（10進なら10）、比較を使わないソート
